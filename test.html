HeART Beat: AR-Powered Heart Rate Relaxation System

October 2024 ‚Äì February 2025

AR Heart Rate Monitoring Interface

Project Overview

HeART Beat is an innovative augmented reality application that combines real-time heart rate monitoring with immersive relaxation experiences. The system detects elevated heart rates through pulse sensors and guides users through calming breathing exercises in a beautiful 3D garden environment.

Key Features

  • Real-time Heart Rate Detection: Integrated pulse sensor with ESP32 microcontroller for continuous monitoring
  • Haptic Feedback System: Adaptive vibration alerts that respond to heart rate changes
  • Immersive AR Environment: Interactive 3D garden with particle effects and spatial anchoring
  • Guided Breathing Exercises: Personalized relaxation routines based on user's stress levels
  • Wearable Integration: Prototyped glove/armband sensors for seamless user experience

Technical Implementation

  • Engineered Bluetooth Low Energy (BLE) connectivity between ESP32 and Unity
  • Developed custom Android plugins to resolve compatibility challenges
  • Created dynamic UI elements with real-time data visualization
  • Implemented spatial anchoring for stable AR experiences
  • Optimized performance for mobile AR platforms (ARKit/ARCore)

Technologies Used

Unity3D C# Scripting Bluetooth Low Energy ESP32 Microcontroller ARKit ARCore 3D Modeling Android Plugins

AdaptiveComfort ‚Äì Smart Shape-Shifting Pillow

October 2024 ‚Äì March 2025

Smart Pillow Control System

Project Overview

AdaptiveComfort is a revolutionary smart pillow that automatically adjusts its firmness based on user posture and sleep position. Using inflatable compartments and advanced sensor technology, it provides personalized comfort without requiring any app interaction.

Key Features

  • Auto-Posture Detection: Recognizes sleeping, reading, and wake-up modes automatically
  • Dynamic Firmness Adjustment: Inflatable Nylon TPU compartments adjust in real-time
  • Multi-Sensor Integration: 3 pressure sensors and gyroscope for accurate position tracking
  • No App Dependency: Fully autonomous operation based on sensor data
  • Optimized Hardware: Reduced complexity from 18 to 2 pumps through smart design

Technical Implementation

  • Developed Arduino-based control logic with real-time sensor processing
  • Implemented TCA9548A I2C multiplexer for managing multiple sensors
  • Engineered airflow optimization system with 6 air pumps and solenoid valves
  • Created iterative prototyping process for hardware optimization
  • Designed fail-safe mechanisms for pump and valve operations

Technologies Used

Arduino UNO I2C Multiplexing MPRLS Pressure Sensors MPU-6050 Gyroscope IoT Prototyping TCA9548A Pneumatic Systems
